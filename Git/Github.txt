2) Generate Token
	1. Go to GitHub and sign in with your account.
	2. Access Developer Settings:
		1. In the upper-right corner of any page, click your profile photo, then click Settings.
		2. In the left sidebar, click Developer settings.
	3. Generate New Token:
		1. In the left sidebar, click Personal access tokens.
		2. Click Generate new token.
	4. Select Scopes:
		1. Give your token a descriptive name in the Note field
		2. Select the scopes or permissions you'd like to grant this token. For example, selecting the repo scope will grant full control of private repositories.
		3. If youâ€™re not sure which scopes to select, you can read more about each scope on the: https://docs.github.com/en/developers/apps/building-oauth-apps/scopes-for-oauth-apps
	5. Generate Token:
		1. After selecting the desired scopes, click Generate token at the bottom of the page.
